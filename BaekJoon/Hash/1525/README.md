## 입력

세 줄에 걸쳐서 표에 채워져 있는 아홉 개의 수가 주어진다. 한 줄에 세 개의 수가 주어지며, 빈 칸은 0으로 나타낸다.

## 출력

첫째 줄에 최소의 이동 횟수를 출력한다. 이동이 불가능한 경우 -1을 출력한다.

## 예제 입력 

### 입력
```
1 0 3
4 2 5
7 8 6
```
### 출력
```
3
```

### 입력
```
3 6 0
8 1 2
7 4 5
```
### 출력
```
-1
```
